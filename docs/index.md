## Usage
Use TypeScript for best in class instellisense.

note this package includes

* tap
* chai
* chai-as-promised
* @types/tap
* @types/chai
* @types/chai-as-promised

```javascript
import {tap, expect} from 'tapbundle' // has typings in place
```
